# KlothoLab — Plataforma de Longevidade

> Plataforma avançada de medicina de precisão e longevidade, similar ao Medvi e TruDiagnostic.

## 🌐 Acesso

**Site:** https://erickrodriguesinvestor-eng.github.io/Univeristy-finding-klotho/

## 🧬 Funcionalidades

- Análise epigenética (Horvath, GrimAge, PhenoAge)
- Dashboard de +200 biomarcadores
- KlothoScore — índice de longevidade
- Protocolo IA personalizado
- Banco de dados IndexedDB com autenticação
- 9 Hallmarks of Aging

## 🏗️ Stack

- HTML5 + CSS3 + JavaScript Vanilla
- IndexedDB (banco de dados local)
- Chart.js (visualizações)
- Design: dark theme + glassmorphism
